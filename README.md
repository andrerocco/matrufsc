# MatrUFSC

MatrUFSC é um planejador de matrícula para cursos de graduação na UFSC. Monte e simule combinações de matérias e turmas para planejar sua matrícula.

Acesse em [matrufsc.github.io](https://matrufsc.github.io/).

## Reconhecimentos
